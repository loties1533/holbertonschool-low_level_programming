# Projet Recursion - Holberton School

## Objectif
Ce projet a pour but de pratiquer la **récursion en langage C**. Chaque exercice consiste à écrire une fonction récursive pour accomplir une tâche spécifique, sans utiliser de boucles ni de fonctions de la bibliothèque standard (sauf `_putchar`).

## Contenu
- Chaque fichier `.c` correspond à un exercice.
- Les fonctions utilisent la récursion pour traiter des chaînes de caractères, des nombres ou d’autres types de données selon l’exercice.
- Les prototypes de toutes les fonctions se trouvent dans le fichier `main.h`.
