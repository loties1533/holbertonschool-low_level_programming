Description

Ce projet consiste à travailler avec les fichiers en C en utilisant les system calls et fonctions de la bibliothèque standard pour ouvrir, lire, écrire et fermer des fichiers. L’objectif est de comprendre le fonctionnement des descripteurs de fichiers, des flags de lecture/écriture et des permissions sous Linux.

Objectifs d’apprentissage

À la fin du projet, vous serez capable de :

Créer, ouvrir, lire, écrire et fermer des fichiers en C

Comprendre et utiliser les descripteurs de fichiers et les trois descripteurs standards (stdin, stdout, stderr)

Utiliser les system calls : open, close, read, write

Comprendre les flags O_RDONLY, O_WRONLY, O_RDWR et les permissions de fichiers

Différencier une fonction d’un system call

Technologies / Ressources

Langage : C

Système : Linux
